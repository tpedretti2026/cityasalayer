# Research Papers Archive

This directory contains papers, articles, and references related to the City as a Layer research.

## Organization
- `by-author/` - Papers organized alphabetically by author
- `by-topic/` - Papers organized by research topic
- `by-year/` - Papers organized chronologically

## Adding Papers
1. Add the PDF or paper details to the appropriate folder
2. Include a README.md or metadata file with citation information
3. Add tags for easy discovery
