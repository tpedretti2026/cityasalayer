# Images

Diagrams, screenshots, and visual assets for the documentation site.

## Image Index

### city-as-layer-framework.jpg
**Figura 1: City as a Layer Framework Diagram**

A comprehensive isometric diagram illustrating the multidimensional framework of the research. Shows the interconnected layers and relationships between:
- **City** - Urban scale and spatial context
- **Stakeholders** - Community and institutional actors
- **Utopia** - Vision and aspirational outcomes
- **Design** - Design as enabling methodology
- **Discipline** - Theoretical and methodological grounding
- **Democracy** - Participatory and governance dimensions
- **Planning** - Urban planning and policy integration
- **Government** - Institutional implementation

This diagram visualizes how design operates across multiple disciplines and stakeholder levels to support urban transformation processes.

### neoliberalism-design-system.jpg

Detailed specification of the design system for the City as a Layer brand. Includes:

This represents the visual identity framework for the research project and its communications.

Images are organized for reference in documentation, presentations, and publications. Each image supports key concepts and visualizations of the research framework.
